Detailed Walkthrough

Step 1: Select Your Input Source

Click the input selector in the top-left corner. Choose from:

  • Webcam — Use your device camera for live video
  • File — Load a video file from your computer
  • Screen — Capture your screen or a specific window
  • URL — Load video from a direct URL

Step 2: Toggle and Adjust Effects

The effect grid shows 16 effects per page across 6 pages (VISION, ACID, GLITCH, OVERLAY, STRAND, MOTION).

  • Click — Toggle effect on/off
  • Drag up/down — Adjust effect intensity
  • Use 1-6 keys — Switch between effect pages

Step 3: Record Your Session

Press R to start/stop recording. Your clip is saved to the Clip Bin and can be exported as WebM or MP4.

SAMPLE SLICER

Chop video into slices for live performance.

  • Click a slice to jump to that position
  • Hold a slice to loop that section
  • Drag slice edges to adjust boundaries
  • Drag clips from Clip Bin into the slicer
Advanced: Slice Controls

Slice Count

Adjust the number of slices using the slice count control. More slices = finer control over playback positions.

Instant Load

Paste a video URL directly into the slicer URL field for instant loading without using the input selector.

Waveform Display

The waveform shows audio amplitude, helping you identify beats and sections visually.

SEQUENCER

Automate any parameter with a multi-track step sequencer.

  • 64-step sequencer with adjustable BPM
  • Drag track handle to any parameter slider to route
  • Multiple track types: Trigger, Gate, Value
  • Euclidean rhythm generator built-in
Advanced: Sequencer Modes

Track Types

  • Trigger — Single pulse per step, good for on/off effects
  • Gate — Sustained value while step is active
  • Value — Smooth parameter automation with interpolation

Euclidean Mode

Generate mathematically distributed rhythms by setting the number of hits and total steps. Great for polyrhythmic patterns.

Parameter Routing

Drag the colored track handle to any parameter slider. The parameter will follow the sequencer track values. Multiple tracks can route to the same parameter for layered automation.

CLIP BIN

Store recordings for instant recall.

  • Recordings automatically saved to Clip Bin
  • Click thumbnail to preview
  • Drag clips to slicer for chopping
  • Export clips as WebM or MP4
Advanced: Clip Management

Storage

Clips are stored in browser IndexedDB. Storage limits depend on your browser and available disk space.

Clearing Clips

Right-click a clip to delete it. Use the clear button to remove all clips (this cannot be undone).

MODULATION

Animate parameters with LFO, Random, Step, and Envelope generators.

  • 4 modulator types: LFO, Random, Step Sequencer, Envelope
  • Drag modulator to any parameter to route
  • Adjust depth, rate, and shape per modulator
  • Stack multiple modulators on the same parameter
Advanced: Modulator Types

LFO (Low Frequency Oscillator)

  • Rate — Speed of oscillation (0.1-20 Hz)
  • Shape — Sine, Triangle, Square, Saw
  • Depth — Amount of modulation applied

Random

  • Rate — How often new random values are generated
  • Smoothing — Interpolation between random values
  • Range — Min/max bounds for random values

Step Sequencer

  • Steps — 4-16 step values
  • Rate — Speed through steps
  • Interpolation — Smooth or stepped transitions

Envelope (ADSR)

  • Attack — Time to reach peak
  • Decay — Time to reach sustain level
  • Sustain — Level while held
  • Release — Time to return to zero

CROSSFADER

Blend between source video and processed effects.

  • Horizontal slider blends SRC (source) and FX (effects)
  • CLEAR button disables all effects instantly
  • BYPASS shows original source without effects
Advanced: Mix Controls

Crossfader Position

  • Full left (SRC) — Original video only
  • Center — 50/50 blend
  • Full right (FX) — Fully processed output

Quick Actions

Use the CLEAR button for instant effect reset during performance. BYPASS temporarily shows the source without changing your effect settings.

TIPS & TRICKS

Power user techniques for advanced workflows.

Sequencer Routing

Drag the colored handle from any sequencer track to any parameter slider. The parameter will follow the track's step values for automated modulation.

Modulation Stacking

Route multiple modulators to the same parameter for complex, layered movement. An LFO + Random creates organic, evolving textures.

Feedback Loops

Combine the FEEDBACK effect with temporal effects (ECHO, SMEAR) for recursive, self-amplifying visuals. Start with low intensities.

Slice Loops

Hold a slice button to loop that section indefinitely. Release to resume normal playback from that position.

Quick A/B Compare

Use the crossfader to quickly compare your source video against the processed output. Great for dialing in subtle effects.

Effect Layering

Effects process in order from top to bottom in the FX chain. Reorder effects by dragging to change the processing sequence.
