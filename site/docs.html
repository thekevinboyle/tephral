Detailed Walkthrough

Step 1: Select Your Input Source

Click the input selector in the top-left corner. Choose from:

  • Webcam — Use your device camera for live video
  • File — Load a video file from your computer
  • Screen — Capture your screen or a specific window
  • URL — Load video from a direct URL

Step 2: Toggle and Adjust Effects

The effect grid shows 16 effects per page across 5 pages (ACID, VISION, GLITCH, STRAND, MOTION).

  • Click — Toggle effect on/off
  • Drag up/down — Adjust effect intensity
  • Use 1-6 keys — Switch between effect pages

Step 3: Record Your Session

Press R to start/stop recording. Your clip is saved to the Clip Bin and can be exported as WebM or MP4.

Sample Slicer

Chop video into slices for live performance and real-time triggering.

The Sample Slicer divides video content into equal segments that can be triggered individually during playback. Similar to sample choppers in audio production, it enables beat-synced video manipulation and live remixing.

Core Interactions

  • Click a slice to jump to that position
  • Hold a slice button to loop that section indefinitely
  • Drag slice edges to adjust boundaries
  • Drop clips from Clip Bin into the slicer for instant loading

Transport Controls

The transport bar provides play/pause, loop toggle, and bypass controls. When bypassed, the slicer shows a prominent "BYPASSED" overlay and all controls are dimmed.

Best Practices

Do
  • Match slice count to musical divisions (4, 8, 16)
  • Use waveform peaks to identify beat positions
  • Pre-load clips for seamless transitions
  • Combine with sequencer for automated triggering
Don't
  • Use too many slices for short clips
  • Trigger rapidly without considering transition smoothness
  • Forget to enable the slicer after loading content

Performance Tips

  • Beat Alignment: Use 8 or 16 slices for 4/4 music to align with bars
  • Loop Points: Hold slice buttons during breakdowns for sustained loops
  • Visual Cueing: The waveform helps anticipate drops and transitions
  • Quick Load: Paste URLs directly into the slicer for instant content loading

SEQUENCER

Automate any parameter with a multi-track step sequencer.

  • 64-step sequencer with adjustable BPM
  • Drag track handle to any parameter slider to route
  • Multiple track types: Trigger, Gate, Value
  • Euclidean rhythm generator built-in
Advanced: Sequencer Modes

Track Types

  • Trigger — Single pulse per step, good for on/off effects
  • Gate — Sustained value while step is active
  • Value — Smooth parameter automation with interpolation

Euclidean Mode

Generate mathematically distributed rhythms by setting the number of hits and total steps. Great for polyrhythmic patterns.

Parameter Routing

Drag the colored track handle to any parameter slider. The parameter will follow the sequencer track values. Multiple tracks can route to the same parameter for layered automation.

CLIP BIN

Store recordings for instant recall.

  • Recordings automatically saved to Clip Bin
  • Click thumbnail to preview
  • Drag clips to slicer for chopping
  • Export clips as WebM or MP4
Advanced: Clip Management

Storage

Clips are stored in browser IndexedDB. Storage limits depend on your browser and available disk space.

Clearing Clips

Right-click a clip to delete it. Use the clear button to remove all clips (this cannot be undone).

Modulation

Animate parameters with LFO, Random, Step, Envelope, and Sample & Hold generators.

The modulation system provides five independent generators that can be routed to any effect parameter. Each modulator runs continuously, producing values that animate parameters over time for organic, evolving visuals.

LFO

Low-frequency oscillator with sine, triangle, square, and saw waveforms. Creates smooth, rhythmic parameter movement.

Random

Generates random values at a configurable rate with optional smoothing for organic variation.

Step

8-step sequencer for precise, repeating patterns. Each step holds a specific value.

Envelope

ADSR envelope generator for attack-decay-sustain-release shapes, triggered by events.

S&H

Sample and Hold captures random values at regular intervals, creating stepped random motion.

Routing

Click the routing icon on any modulator card, then click a parameter in the Effects Lane to create a connection. Active routings are shown as colored lines between modulators and parameters.

Best Practices

Do
  • Start with subtle depth values and increase gradually
  • Sync LFO rates to your content's tempo
  • Stack multiple modulators for complex movement
  • Use S&H for glitchy, unpredictable variations
Don't
  • Route too many modulators to the same parameter at high depth
  • Use very fast LFO rates on sensitive parameters
  • Forget to enable the modulator after routing

Creative Techniques

  • Polyrhythm: Route LFO and Step at different rates for evolving patterns
  • Organic Drift: Random + Envelope creates natural, breathing movement
  • Glitch Bursts: S&H with fast rate for digital chaos
  • Cross-Modulation: Route to multiple related parameters simultaneously

Crossfader

Blend between source video and processed effects with DJ-style control.

The crossfader provides real-time blending between your source video (SRC) and the fully processed output (FX). This enables smooth transitions and A/B comparison during live performance.

Crossfader Positions

  • Full Left (SRC) Shows original video only, all effects bypassed
  • Center 50/50 blend between source and processed
  • Full Right (FX) Fully processed output with all enabled effects

Quick Actions

  • CLEAR Instantly disables all effects, returns to clean source
  • BYPASS Temporarily shows source without changing effect states

Best Practices

Do
  • Use for dramatic effect reveals during drops
  • BYPASS to quickly check your source is correct
  • Route LFO to crossfader for pulsing effect intensity
  • CLEAR as an emergency reset during performance
Don't
  • Confuse CLEAR (disables effects) with BYPASS (temporary)
  • Leave crossfader at SRC when trying to adjust effects

Performance Techniques

  • Effect Punch: Start at SRC, slam to FX on beat hits
  • Gradual Build: Slowly move from SRC to FX during crescendos
  • A/B Compare: Toggle between positions to fine-tune effect intensity
Property Type Default Range
position number 1.0 0 (SRC) – 1 (FX)
curve enum linear linear, equal-power

Behavior

  • GPU-accelerated: Blending happens in the shader pipeline
  • Zero latency: Position changes are instant
  • Modulatable: Can be targeted by any modulation source

TIPS & TRICKS

Power user techniques for advanced workflows.

Sequencer Routing

Drag the colored handle from any sequencer track to any parameter slider. The parameter will follow the track's step values for automated modulation.

Modulation Stacking

Route multiple modulators to the same parameter for complex, layered movement. An LFO + Random creates organic, evolving textures.

Feedback Loops

Combine the FEEDBACK effect with temporal effects (ECHO, SMEAR) for recursive, self-amplifying visuals. Start with low intensities.

Slice Loops

Hold a slice button to loop that section indefinitely. Release to resume normal playback from that position.

Quick A/B Compare

Use the crossfader to quickly compare your source video against the processed output. Great for dialing in subtle effects.

Effect Layering

Effects process in order from top to bottom in the FX chain. Reorder effects by dragging to change the processing sequence.
